# 💬 Modern AI Chat UI (Assignment 3)

Welcome to my Assignment 3 frontend project.

This project presents a fully responsive and modern chat interface created using **HTML5, CSS3, JavaScript, jQuery, and Bootstrap 5**. The main goal was to replicate the clean and intuitive user experience seen in tools like ChatGPT and Claude, without using any backend services.

## ✨ What's Included?

To make the interface feel like a real AI product, the UI includes the following features:
- **ChatGPT-Inspired Layout**: A centered conversation view with a floating input area and a responsive collapsible sidebar.
- **Claude-Like Animations**: Messages animate smoothly into view, and AI output appears word-by-word with a subtle fade effect for realistic streaming.
- **Model Selection Menu**: Switch between simulated models such as ChatGPT, Google Gemini, Ollama, Groq, Hugging Face, and Cohere from the top header.
- **Functional File Attachments**: Click the paperclip icon to choose files from your system and preview them as chips before sending.
- **Dark Mode Support**: Use the sidebar toggle to transition the interface into a dark theme powered by CSS variables.
- **Export Chat Feature**: Download the conversation instantly as a well-formatted text file.

## 🚀 How to Run It
This project is frontend-only, so getting started is very easy:
1. Open `index.html` in any modern browser (Chrome, Edge, Firefox, or Safari).
2. Send messages, attach files, switch models, and test the available UI interactions.

## 🛠️ Tech Stack
- **HTML5**: Provides semantic and accessible page structure.
- **CSS3 / Bootstrap 5**: Ensures responsive behavior across mobile and desktop screen sizes.
- **JavaScript / jQuery**: Handles core interactivity such as auto-resizing input, simulated response timing, and message rendering.
- **FontAwesome & Google Fonts**: Uses Inter typography and vector icons for a polished visual style.
